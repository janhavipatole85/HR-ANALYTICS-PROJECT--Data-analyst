# 📊 HR Analytics Dashboard

## 🚀 Project Overview
This HR Analytics project focuses on analyzing employee data to uncover valuable workforce insights. The project helps HR teams understand employee attrition, workforce demographics, job satisfaction, and performance trends to support data-driven decision-making.

## 🎯 Business Objective
The primary objective of this project is to:
- Analyze employee attrition patterns
- Identify factors affecting employee retention
- Monitor workforce demographics
- Improve HR decision-making through data visualization

## 🛠️ Tools & Technologies
- SQL
- Power BI
- Tableau
- Microsoft Excel

## 📂 Project Files
- SQL Queries

- Dashboard Screenshots
- HR ANALYTICS (dataset) 

## 📊 Key KPIs
- Total Employees
- Attrition Rate
- Average Age
- Average Monthly Income
- Job Satisfaction Score
- Department-wise Attrition
- Gender Distribution

## 🔍 Key Insights
- Identified departments with the highest attrition rates.
- Analyzed employee demographics by age, gender, and education.
- Evaluated the impact of job satisfaction on employee retention.
- Compared workforce trends across different departments.
